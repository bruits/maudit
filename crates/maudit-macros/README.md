# maudit-macros

Procedural macros exposed by the [Maudit](https://github.com/bruits/maudit/blob/main/crates/maudit/README.md) library, including the `#[route]` attribute used to declare pages.

Part of the [Maudit](https://github.com/bruits/maudit) monorepo — see the top-level [README](https://github.com/bruits/maudit/blob/main/README.md) and [CONTRIBUTING.md](https://github.com/bruits/maudit/blob/main/CONTRIBUTING.md) for project-wide documentation.
